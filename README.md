## Getting Started

#### Database

For running the database you should have **Docker compose** installed then just run

- **Starting the DB**
  
  ```bash
  docker compose up -d
  ```

- **Stopping DB**
  
  ```bash
  docker compose down
  ```

- **Stopping and deleting volumes**
  
  ```bash
  docker compose down -v
  ```
